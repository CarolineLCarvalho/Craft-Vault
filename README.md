🎮 CraftVault 🎮 

Sistema de gerenciamento de biblioteca de jogos desenvolvido com React, inspirado em plataformas como Steam, GamePass e launchers de games, utilizando um visual temático inspirado no universo do Minecraft.

📌 Sobre o Projeto

O projeto foi desenvolvido com o objetivo de praticar os principais conceitos do React, como:

Componentização
JSX
Props
Hooks
Renderização dinâmica
Navegação entre páginas
Consumo de API
Manipulação de estados
Persistência de dados

A aplicação permite que o usuário:

✅ Busque jogos reais através da API RAWG
✅ Cadastre jogos na biblioteca
✅ Visualize jogos em cards estilizados
✅ Remova jogos cadastrados
✅ Navegue entre páginas utilizando React Router

🛠 Tecnologias Utilizadas
React
JSX
CSS3
React Router DOM
JavaScript
Fetch API
RAWG API
LocalStorage
Vite

Toda a estilização foi feita com:
CSS puro.

🎮 Funcionalidades
📥 Cadastro de Jogos

O usuário pode selecionar jogos reais consumidos da API RAWG.

Ao selecionar um jogo:
nome
imagem
gênero

são preenchidos automaticamente.

📚 Biblioteca de Jogos

Os jogos cadastrados são exibidos em formato de cards dinâmicos.

Cada card possui:
imagem do jogo
nome
gênero
status
botão de remoção

❌ Remoção de Jogos
O usuário pode remover jogos da biblioteca utilizando o botão “Remover”.

A remoção:

atualiza a interface
atualiza o localStorage
remove o jogo permanentemente da biblioteca local

💾 Persistência de Dados

Os dados são armazenados utilizando:

localStorage

Isso garante que:

os jogos permaneçam salvos
mesmo após atualizar a página
🌐 API Utilizada
RAWG API

API utilizada para buscar informações reais dos jogos.
Site oficial: RAWG API.
